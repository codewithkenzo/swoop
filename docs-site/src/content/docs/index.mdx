## What is Swoop?

Swoop eliminates the complexity of document intelligence by providing a single API that transforms any document into structured, searchable, and actionable data.

### Key Features

- **⚡ Lightning Fast**: Sub-second document processing with Rust performance
- **🧠 AI-Powered**: Automatic categorization, entity extraction, and quality scoring
- **🔍 Hybrid Search**: Combines keyword and semantic search for best results
- **💬 Document Chat**: Ask questions about your documents with AI
- **🌐 Web Crawling**: Intelligent web scraping with robots.txt compliance
- **🛡️ Enterprise Ready**: Production-grade security and monitoring

## Getting Started

Ready to transform your documents? Here's how to get started:

### 1. Install Swoop

```bash
curl -fsSL https://install.swoop.dev | bash
```

### 2. Upload Your First Document

```bash
curl -X POST "http://localhost:8080/api/documents/upload" \
  -H "Authorization: Bearer your-api-key" \
  -F "file=@document.pdf"
```

### 3. Search Your Content

```bash
curl "http://localhost:8080/api/search?q=artificial+intelligence" \
  -H "Authorization: Bearer your-api-key"
```

### 4. Chat with Your Documents

```bash
curl -X POST "http://localhost:8080/api/documents/doc-123/chat" \
  -H "Authorization: Bearer your-api-key" \
  -d '{"message":"What are the main points?"}'
```
